# PWA Screenshots

This directory contains screenshots for the Progressive Web App (PWA) manifest.

## Required Screenshots

- **desktop.png** - Desktop/wide view (1280x720px or similar)
- **mobile.png** - Mobile/narrow view (750x1334px or similar)

## Purpose

Screenshots are shown in app stores and install prompts to give users a preview of your app.

## Current Status

⚠️ **Placeholder directory** - Add actual app screenshots before production deployment.

## How to Create

1. Open your app in a browser
2. Navigate to key screens (dashboard, transactions, budgets)
3. Take screenshots at recommended resolutions
4. Save as PNG files in this directory
5. Update manifest.webmanifest if you change filenames

## Specifications

- **Desktop**: 1280x720px or wider (form_factor: "wide")
- **Mobile**: 750x1334px or similar mobile aspect ratio (form_factor: "narrow")
- **Format**: PNG
- **Content**: Show actual app UI, not mockups
